# Binary Search
Binary Search involves repeatedly dividing a sorted list in two until the sought after value is found.  The list must be sorted.


## Time Complexity
### Best Case
first number checked is number sought  
  O(1)  
  Ω(1)  
  Θ(1)  
 
### Worst Case
number sought is not present  
  O(logn)  
  Ω(logn)  
  Θ(logn)
